# 🎯 Project 3: AI Recommendation Logic
### DecodeLab Internship

---

## 📌 About This Project
A product recommendation system that suggests items to users
based on their preferences and buying patterns.
Built using two methods: Rule-Based Logic and Cosine Similarity.

---

## 🎯 What It Does
- Takes user input — their interest or a product they liked
- Method 1: Rule-Based — recommends by category and budget
- Method 2: Similarity-Based — finds similar products using cosine similarity
- Shows a ranked list of recommended products
- Runs in interactive mode so user can type their choice

---

## 🛠️ Skills Used
- Python logic building
- Pattern matching
- Cosine similarity (Scikit-learn)
- Pandas data aggregation
- Recommendation system concepts

---

## 📦 Dataset
File: Product-Sales-Region.xlsx
Used to compute product similarity from real order history data.

---

## 🚀 How to Run
1. Open the folder in VS Code
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `source venv/Scripts/activate` (Git Bash) or `venv\Scripts\activate` (Command Prompt)
4. Open `Project3_AI_Recommendation.ipynb` in VS Code
5. Select the `venv` kernel
6. Run all cells one by one
7. When prompted, choose Mode 1 or Mode 2 and type your preference
